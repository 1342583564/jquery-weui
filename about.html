关于 jQuery WeUI

WeUI 目前只包含 CSS 代码，官方短期内不会出JS版本，所以我做了一个jQuery版本的方便使用。

jQuery WeUI 的组件分两部分：

  • 基础组件：weui官方的组件，使用了weui官方的css
  • 拓展组件：jquery weui 专属的组件

有任何Bug或者建议，请在 Github 上提 Issue 给我。或者加入 jQuery WeUI 专属QQ群 546452237.

后续计划

jQuery WeUI 的主要计划是增强现有组件的功能，并根据需求添加新的组件。但是jQuery WeUI 并不打算解决整个SPA的问题，只是想专注做一些常用的UI控件，所以路由器并不在 jQuery WeUI 的计划之中。

jQuery WeUI 的定位正如 jQuery 的定位一样：做一把锋利的小刀，而不是做一个笨重的大炮。

另外我有计划做一个基于 Vue 或者 Angular2 的版本，这个版本会依赖框架的路由来解决单页应用的问题，有进展的话会随时通知大家。
