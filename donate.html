帮助 jQuery WeUI

jQuery WeUI 是 MIT 协议的免费开源项目，不会收取任何费用。

作者和大家一样也是一枚程序员，白天上班，业余时间维护这个项目需要耗费不少精力，如果jQuery WeUI 帮到了你，你可以通过以下任意方式来帮她变得更好：

  • 通过 Issue 提出自己开发过程中碰到的问题。
  • 通过 Pull Request 贡献自己的代码，成为此项目的开发者之一。
  • 在 github 上给点击 Star 给我加星，更多的星星是我前进的动力。
  • 扫描下方二维码，给作者捐助，收集到的捐款项将全部展示在此页面下方的捐助墙上。

我本人从未想过因为这个项目而获得直接多少金钱收益，然而从上个月上线捐助功能之后，确实不断收到捐助。虽然全部加起来可能也比不上我一天的工资，然而我还是非常感动的，我看到了确实有一部分人会愿意为免费的开源项目捐款，我也相信我们的整个互联网的环境会变得越来越健康的，每一份付出也是会得到应有的回报的。 - 言川
