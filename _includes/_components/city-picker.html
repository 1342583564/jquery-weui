参数

除了 `Picker` 的全部参数可用之外，`city-picker` 还有如下参数可以配置:

参数名 默认值 说明
showDistrict true 是否显示地区

比如如下设置可以不显示地区（只选择省市):

{% highlight js %} //禁用地区选择 $("#city-picker").cityPicker({ showDistrict: false }); {% endhighlight %}

你可以通过 $.fn.cityPicker.prototype.defaults 来改变默认值。
